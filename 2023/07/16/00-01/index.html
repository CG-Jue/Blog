    题目

    首先是一个server.py

    明显的是e很小，一开始思路是低指数加密，but 这个更小（一般为3）

    还给了个在线环境

    想了半天没搞明白，

    后来学长提示这样可以获取任意组n和c

    意味着有多组 n、c不同，but明文m，e相同，也就是低加密指数广播攻击

    中国剩余定理

    附赠一个脚本加一个blog

    以及生蚝✌养殖场

    import gmpy2
from functools import reduce
from Crypto.Util.number import long_to_bytes

# 中国剩余定理的算法
def CRT(cipher, n):
    N = reduce(lambda x, y: x * y, (i for i in n))
    result = 0
    data = zip(cipher, n)
    for ci, ni in data:
        Ni = N // ni
        di = gmpy2.invert(Ni, ni)
        result += ci * Ni * di
    return result % N, N

# 读入 e, n, c
e = 111#这里e看题目，如果给了按给的，比如本题就是111,如果没给就是n和c的组数
n = []
c = []

x, N = CRT(c, n)

m = gmpy2.iroot(gmpy2.mpz(x), e)[0]

print(m)
print(long_to_bytes(m))

    分析

    与一般不同点在于e取111,也就是说，如果手搓111次的话（也不是不行，甚至以我写脚本的速度还要稍微快一点）

    but当e三四百的时候就很寄，能学写脚本，谁愿意当脚本小子呢

    我们这里可以用到pwntools的环境交互（学习的时候遇到了其他的交互，但是pwntools对于ctfer用的还是比较多的，就学了pwntools，毕竟对于一个pwn手来说pwntools是基操）

    remote('url',port)#远程交互并且函数返回给remote对象，我们可以赋值给a。
eg:a=remotr('url',port)

    接下俩可以用send或recv操作

    • send(payload) 发送payload
    • sendline(payload) 发送payload，并进行换行（末尾\n）
    • sendafter(some_string, payload) 接收到 some_string 后, 发送你的 payload
    • recvn(N) 接受 N(数字) 字符
    • recvline() 接收一行输出
    • recvlines(N) 接收 N(数字) 行输出
    • recvuntil(some_string) 接收到 some_string 为止

    这里是返回一行n=什么什么和一行c=什么什么

    可以用如下来分别接受两行，用来区分开n和c然后利用上面的脚本解题

    line1 = a.recvline()
line2 = a.recvline()

    还有一些微操去调整数据格式，以便于符合脚本就不多说了（当然不是懒得改脚本了）😖

    😵‍💫最终exp

    from pwn import *
import gmpy2
from functools import reduce
from Crypto.Util.number import long_to_bytes
n=[]
c=[]
for i in range(111):	
    conn = remote("url",10000)
    line1 = conn.recvline()
    data1 = line1.strip()
    new1=str(data1)[6:-1]
    n.append(int(new1))
    line2 = conn.recvline()
    data2 = line2.strip()
    new2=str(data2)[6:-1]
    c.append(int(new2))
    conn.close()
def CRT(cipher, n):

    data = zip(cipher, n)
    for ci, ni in data:
        Ni = N // ni

        result += ci * Ni * di
    return result % N, N
e = 111

    Sage exp

    from Crypto.Util.number import *
from pwn import *
from tqdm import trange
n = []
c = []
for i in trange(111):
    io = remote('url', '10000')
    n.append(eval(io.recvline().decode()[3:]))
    c.append(eval(io.recvline().decode()[3:]))
    io.close()
    sleep(0.1)

e = 111
ms = crt(c, n)
R.<x> = PolynomialRing(ZZ)
f = x^e - ms
m = f.roots()[0][0]
print(long_to_bytes(m))
